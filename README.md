## 📖 Descripción

Este proyecto permite explorar datos históricos sobre ataques de tiburones a nivel global mediante un tablero interactivo. Está diseñado para aquellos que deseen visualizar patrones y tendencias en incidentes históricos.

---

## ✨ Características

- ✅ Mapas interactivos en 2D y 3D.
- ✅ Filtros dinámicos por país y tipo de ataque.
- ✅ Resumen de métricas clave.
- ✅ Tabla de datos detallada.
- ✅ Visualización optimizada para grandes volúmenes de datos.

## ⚙️ Instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/tu-usuario/PROYECTO-RADAR-TIBUROES.git

# 2. Entrar en la carpeta
cd PROYECTO-RADAR-TIBUROES

# 3. Crear y activar un entorno virtual
python3 -m venv .venv
source .venv/bin/activate

# 4. Instalar las dependencias
pip install -r requirements.txt
```

---

## 💡 Uso

1. Ejecuta la aplicación:
   ```bash
   streamlit run dashboard_tiburones_ejercicio.py
   ```
2. Abre el enlace generado en tu navegador para interactuar con el tablero.
